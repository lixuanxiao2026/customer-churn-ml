# Raw Data

## Dataset: Churn BigML 20

**File:** `churn-bigml-20_raw.csv`

**Source:** BigML Churn Modeling Benchmark (telecommunications customer churn)

**Description:** Original, unmodified dataset for customer churn prediction. Contains customer attributes and usage metrics with binary churn target.

**If dataset is not present:** Download from [Kaggle - Telco Customer Churn](https://www.kaggle.com/datasets/mnassar/telco-customer-churn) or [BigML](https://bigml.com/datasets) and place in this folder.

**Requirements:**
- No modifications to original data
- Keep this folder as the single source of truth for raw inputs
